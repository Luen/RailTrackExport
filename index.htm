Railtrails GPX Exporter

About

The Railtrails GPX Exporter is a simple tool that allows you to extract trail data from the Railtrails website and export it as a GPX file. GPX files are widely used for storing GPS track data and can be imported into various mapping and navigation applications.

How to Use

To use this tool, simply drag the following link to your bookmarks bar. Then, whenever you're on a Railtrails page with a trail map, click the bookmarklet to download the trail data as a GPX file.

Railtrails GPX Exporter

Instructions

  1. Drag the "Railtrails GPX Exporter" link above to your bookmarks bar.
  2. Go to a trail page on the Railtrails website.
  3. Click the bookmarklet in your bookmarks bar.
  4. Your GPX file will be automatically downloaded.
